Windows Installation

 

Requirements
  1. Fast internet connection.
  2. Admin rights on the machine.
  3. At least 10GB available.
  4. Know how to use Windows File Explorer.
  5. Know how to run .exe files as an administrator.
  6. Know how to use Windows Powershell as an administrator.
  7. Know how to use Windows Command Prompt

 

Game - Installation Steps
  1. Download installer files (download all of them to the same folder!).
  2. Run FFXIFullSetup_US.part1.exe to decompress installer files
  3. Run FFXISETUP.exe after decompression completes to begin installing Final Fantasy XI (FFXI) and PlayOnline (POL).
  4. To avoid possible future issues with allowing FFXI to access/modify its own files, it is best to install FFXI and POL to a folder of its own on your computer.
  5. Make sure to install everything that prompts on the installer.
  6. Once you are done with the installation we can move to the setup phase.

 

PlayOnline Setup
  • Open PlayOnline Viewer and go through the prompts using your PlayOnline account you have registered previously. If you don't have an account, you can just enter a faux one that is 4 Letters, followed by 4 numbers (i.e., you can enter ABCD1234 if you don't have one).
  • Close PlayOnline.strong>

 

Game - Update Steps (semi-automatic)
  • Download the update_xi.ps1 script (note the location it was downloaded to, you'll need that shortly).
  • Open Powershell as an administrator.
  • Run powershell.exe -ExecutionPolicy Bypass to allow running the script.
  • Navigate to the folder that update_xi.ps1 was downloaded to.
  • Run .\update_xi.ps1. Playonline should start automatically after this script completes.
  • NOTE: If you encounter errors/issues with the above, STOP HERE and skip down to the steps in the "Game - Update Steps (manual)" section below.
  • If Playonline didn't automatically start, open Playonline once more and choose Check Files.
  • On the next screen, make sure you change the Playonline-compatible software choice to FINAL FANTASY XI.
  • Click on Check Files to repair the files.
  • It should tell that a file is missing and that it needs repair, followed by a long download and install.
  • Once it's done repairing files, close Playonline. We can move on to the “Game - First Run” step.

 

Game - Update Steps (manual)
  • Browse using “File Explorer” and go to C:\Program Files (x86)\PlayOnline\SquareEnix\FINAL FANTASY XI
  • Create a new file called file.txt
  • Open file.txt and populate it with the info below, then save it. Make sure to also add a 3rd empty line.
123456:654321:FTABLE.DAT
::
  • Open Playonline once more and choose Check Files.
  • On the next screen, make sure you change the Playonline-compatible software choice to FINAL FANTASY XI.
  • Click on Check Files to repair the files.
  • It should tell that a file is missing and that it needs repair, followed by a long download and install.
  • Once it's done repairing files, close Playonline. We can move on to the “Game - First Run” step.

 

Game - First Run
  1. To run FFXI for the first time (and all subsequent times), you have a choice of three "bootloaders" that are used to load the game and connect to the XIWeb game server:

 

XiLoader (the barebones bootloader, loads the game and nothing else, best for testing out issues before reporting them)
  1. Download XiLoader.exe into a directory of its own.
  2. Open Command Prompt Administrative-level privileges.
  3. Find the folder where you downloaded XILoader.exe and run it as follows
xiloader.exe --server xiweb.catsangel.com

 

Ashita (a bootloader that includes a variety of addons/plugins to enhance your playing experience) (more geared towards players on private servers like XIWeb)
  1. Download Ashita.exe into a directory of its own.
  2. Run Ashita.exe with Administrative-level privileges. Ashita will download all other necessary files and then present you with the Ashita launcher
  3. Download XiLoader.exe into the ffxi-bootmod sub-folder of your Ashita folder.
  4. In the Ashita launcher, click on the + icon in the bottom-left of the Ashita Launcher window.
  5. Click on the New Configuration that was just created then click on the Pencil icon (that is to the right of the + icon) in the bottom-left of the Ashita Launcher window.
  6. Give your new configuration a name (xiweb or whatever name you wish).
  7. For the File: line, use: .\\ffxi-bootmod\\xiloader.exe
  8. For the Command: line, use: --server xiweb.catsangel.com
  9. Click on the icon in the bottom-center of the window to save your changes (the icon looks like a sheet of paper with a checkmark).
  10. Double-click on the name that you gave for the configuration you just made (in the Ashita launcher) to launch the bootloader.
  11. If this is your first time playing on XIWeb, you will need to create an account and a password
    12. and use such to connect to XIWeb.
  12. If you already have an account on XIWeb, you need to use the account and password you created to connect to XIWeb.
Windower (a bootloader that includes a variety of addons/plugins to enhance your playing experience) (more geared towards players on retail FFXI but can also be used for private servers like XIWeb)
  1. Download Windower.msi into a directory of its own.
  2. Run Windower.msi with Administrative-level privileges. Windower will install to a folder of your choice (best to use the folder you downloaded Windower.msi into)
  3. Navigate to the folder you installed Windower into and run Windower.exe Administrative-level privileges. Windower will download all other necessary files and then present you with the Windower launcher.
  4. In the Windower Launcher, click on the + in the bottom-left of the Ashita Launcher window to create a new profile.
  5. Give the new profile a name (click on the New Profile 1 and change it to a new name of your choice) and then click on the left-pointing arrow in the upper-left corner of the profile. (Make sure to remember the name you gave the profile)
  6. Close the Windower launcher.
  7. Download XiLoader.exe into the PlayOnlineViewer sub-folder of the SquareEnix subfolder of your PlayOnline folder on your computer (created in the Game - Installation steps at the top of this page) .
  8. Navigate to the folder that you installed Windower into and open settings.xml with some type of text editor, such as Windows Notepad or Notepad++.
  9. You'll need to add the XiLoader.exe to every profile (including the profile you just created) you see in settings.xml, as well as the with your server information and login.
  10. Here is an example profile (you can copy and paste the below code block into the file:
    11. 
    Insert
    --server xiweb.catsangel.com
    DRIVE_LETTER:\path\to\xiloader.exe
  11. Note the "path\to\" portion in the section cannot contain spaces and cannot be a relative path.
  12. An example of the path is: C:\PlayOnline\SquareEnix\PlayOnlineViewer\
  13. Make sure you save changes to the file and then exit out of the text editor completely.
  14. Launch Windower again.
  15. Double-click on the name that you gave for the configuration you just made (in the Windower launcher) to launch the bootloader.
  16. If this is your first time playing on XIWeb, you will need to create an account and a password
    17. and use such to connect to XIWeb.
  17. If you already have an account on XIWeb, you need to use the account and password you created to connect to XIWeb.
