Linux Installation

 

  1. Install Wine or Proton.
  2. Download installer files.
  3. Download xiloader.exe.
  4. Decompress installer files.exe
  5. Run the setup file that gets decompressed.
  6. Run Playonline once, register for an account or use an existing one, or enter ABCD1234 if you don't have one. When done, close it.
  7. Go to C:\Program Files(x86)\Square Enix\Playonline\Final Fantasy XI and delete the file FTABLE.DAT
  8. Create a new file called file.txt inside C:\Program Files (x86)\PlayOnline\SquareEnix\FINAL FANTASY XI
  9. Open file.txt and populate it with the info below. Make sure to also add a 3rd empty line.
123456:654321:FTABLE.DAT
::
  1. Run Playonline once more and repair the files. Once its completed close it.
  2. Run xiloader.exe --server xiweb.catsangel.com
  3. Celebrate freedom.

 

Windows Installation

 

Requirements
  1. Fast internet connection.
  2. Admin rights on the machine.
  3. At least 10GB available.
  4. Know how to use Windows Explorer.
  5. Know how to run .exe files as an administrator.
  6. Know how to use Command Prompt

 

Game - Installation Steps
  1. Download the installer files.
  2. Run the file with extension .exe to decompress the installer.
  3. Launch the setup file inside the decompressed folder.
  4. Make sure to install everything that prompts on the installer.
  5. Once you are done with the installation we can move to the setup phase.

 

Game - Update Steps
  • Open Playonline Viewer, and go through the prompts using your Playonline account you have registered previously. (If you don't have an account, you can enter a faux one that is 4 Letters, followed by 4 numbers.
  • Close Playonline.
  • Browse using “File Explorer” and go to C:\Program Files (x86)\PlayOnline\SquareEnix\FINAL FANTASY XI
  • Delete the FTABLE.DAT, and create a new file called file.txt
  • Open file.txt and populate it with the info below, then save it. Make sure to also add a 3rd empty line.
123456:654321:FTABLE.DAT
::
  • Open Playonline and select “Repair Files”
  • Select FFXI, and repair the files
  • It should tell that a file is missing, and that it needs repair, followed by a long download and install.
  • Once it's done repairing files we can move on the launch step

 

Game - First Run
  1. Download xiloader.exe
  2. Open Command Prompt as an Administrator
  3. Find the folder where the file is and run it as follows
xiloader.exe --server xiweb.catsangel.com
