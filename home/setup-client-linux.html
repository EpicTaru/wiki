Linux Installation

 

  1. Install Wine or Proton.
  2. Download installer files (download all of them to the same folder!).
  3. Download xiloader.exe.
  4. Run FFXIFullSetup_US.part1.exe to decompress the installer files
  5. Run FFXISETUP.exe after the decompression completes.
  6. Make sure to install everything that prompts on the installer.
  7. Open PlayOnline and go through the prompts using your PlayOnline account you have registered previously. If you don't have an account, you can just enter a faux one that is 4 Letters, followed by 4 numbers (i.e., you can enter ABCD1234 if you don't have one).
  8. Let Playonline update itself if it prompts. When done, close Playonline.
  9. Create a new file called file.txt inside C:\Program Files (x86)\PlayOnline\SquareEnix\FINAL FANTASY XI
  10. Open file.txt and populate it with the info below. Make sure to also add a 3rd empty line.
123456:654321:FTABLE.DAT
::
  1. Open Playonline once more and choose Check Files.
  2. On the next screen, make sure you change the Playonline-compatible software choice to FINAL FANTASY XI.
  3. Click on Check Files to repair the files.
  4. It should tell that a file is missing and that it needs repair, followed by a long download and install.
  5. Once it's done repairing files, close Playonline. We can move on to the “Game - First Run” step.
