Requirements
  1. Fast internet connection.
  2. Admin rights on the machine.
  3. At least 10GB available.
  4. Know how to use Windows Explorer.
  5. Know how to run .exe files as an administrator.
  6. Know how to use Command Prompt

 

Installation Steps
  1. Download the installer files in here
  2. Run the file with extension .exe to decompress the installer.
  3. Launch the setup file inside the decompressed folder.
  4. Make sure to install everything that prompts on the installer.
  5. Once you are done with the installation we can move to the setup phase.

 

Setup
  1. Open Playonline Viewer, and go through the prompts using your Playonline account you have registered previously. (If you don't have an account, you can enter a faux one that is 4 Letters, followed by 4 numbers.
  2. Close Playonline.
  3. Browse using “File Explorer” and go to C:\Program Files(x86)\Square Enix\Playonline\Final FantasyXI
  4. Delete the FTABLE.DAT, and create file called file.txt
  5. On file.txt, enter as the first line the following
    123456:654321:./FTABLE.DAT
    ::
  6. Open Playonline and select “Repair Files”
  7. Select FFXI, and repair the files
